Use cases:
UC1: Set a timed lock
    1. Open the executable .exe
    2. Select date time to unlock
        2.1 Select date from a monthly calendar
        2.2 Select a specific time from a list of times of the day spaced by .5 hour
    3. Select which program to lock
    4. Press "LOCK"
